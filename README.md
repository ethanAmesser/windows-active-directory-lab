# Windows Active Directory Infrastructure Lab

## Overview
This project demonstrates the deployment and configuration of a Windows Server Active Directory environment. The lab focuses on domain services, centralized user management, group policy implementation, and web server configuration within a controlled enterprise network environment.

## Technologies Used
- Windows Server
- Active Directory Domain Services
- Group Policy
- IIS Web Server
- Windows Networking

## Skills Demonstrated
- Active Directory domain configuration
- User and group management
- Group Policy implementation
- Windows server administration
- Network service configuration

## Key Activities
- Deployed a Windows Server domain controller within a virtual lab environment.
- Configured Active Directory Domain Services and created a new domain.
- Created and managed user accounts and security groups.
- Implemented Group Policy Objects to enforce security and system policies.
- Deployed and configured an IIS web server to host internal services.
